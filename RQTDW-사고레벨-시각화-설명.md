# RQTDW© 사고 레벨 시각화 그래프 기능

## 📊 개요

Write(W) 단계에 **사용자의 사고 성장을 시각적으로 확인할 수 있는 그래프 시스템**을 추가했습니다.

---

## 🎨 시각화 구성 요소

### 1️⃣ 레이더 차트 (Radar Chart)

**6가지 사고 능력을 한눈에 비교**

```
       질문 깊이
          /\
         /  \
창의적연결    관점다양성
    |          |
실행력    구조화능력
    \        /
      \    /
    비판적사고
```

#### 측정 항목 (6가지)
1. **질문 깊이**: 표면 → 본질 질문으로 발전하는 능력
2. **관점 다양성**: 다면적 시각을 확보하는 능력
3. **구조화 능력**: 프레임워크를 활용한 사고 정리
4. **비판적 사고**: 전제를 의심하고 검증하는 능력
5. **창의적 연결**: 서로 다른 개념을 연결하는 능력
6. **실행력**: 통찰을 행동으로 전환하는 능력

#### 2개 데이터셋 비교
- **파란색 영역**: 현재 사용자 레벨 (실선)
- **회색 점선**: 평균 레벨 6.0 (기준선)

#### 인터랙티브 기능
- 마우스 오버 시 정확한 점수 표시
- 각 능력별 세부 점수 확인 가능

---

### 2️⃣ 프로그레스 바 (Progress Bars)

**3대 핵심 지표를 숫자+바 형태로 표시**

```
┌─────────────────────────┐
│ 질문 깊이       7.5/10  │
│ ████████████░░░ 75%     │
└─────────────────────────┘

┌─────────────────────────┐
│ 관점 다양성     8.0/10  │
│ █████████████░░ 80%     │
└─────────────────────────┘

┌─────────────────────────┐
│ 구조화 능력     7.0/10  │
│ ███████████░░░░ 70%     │
└─────────────────────────┘
```

#### 특징
- **애니메이션**: 0% → 실제 점수까지 1초간 증가
- **컬러 구분**: 파란색 / 초록색 / 보라색
- **transition-all**: 부드러운 애니메이션 효과

---

### 3️⃣ 성장 추이 차트 (Growth Trend Chart)

**최근 5회 훈련의 성장 곡선**

```
10 ┤                              
 9 ┤                        ●────  관점
 8 ┤                  ●───●        
 7 ┤            ●───●       ●───   질문
 6 ┤      ●───●               ●── 구조
 5 ┤●───●                         
 0 └─────────────────────────────
   1회  2회  3회  4회  5회(현재)
```

#### 3가지 능력 추적
- **파란선**: 질문 깊이
- **초록선**: 관점 다양성
- **보라선**: 구조화 능력

#### 데이터
```javascript
1회차 → 2회차 → 3회차 → 4회차 → 5회차(현재)
질문:  5.5 → 6.0 → 6.5 → 7.0 → [현재점수]
관점:  6.0 → 6.5 → 7.0 → 7.5 → [현재점수]
구조:  5.0 → 5.5 → 6.0 → 6.5 → [현재점수]
```

#### 인터랙티브 기능
- Fill 효과: 영역 아래 색칠
- Tension 0.3: 부드러운 곡선
- 포인트 호버: 회차별 정확한 점수

---

### 4️⃣ AI 종합 평가

**AI가 분석한 강점과 개선점**

```
┌────────────────────────────────┐
│ 🎯 AI 종합 평가                 │
├────────────────────────────────┤
│ 성장 포인트:                    │
│ 질문을 재구성하는 능력이        │
│ 뛰어납니다. 다음에는 더 다양한  │
│ 프레임워크를 활용해보세요.      │
│                                 │
│ ✓ 질문 깊이 우수                │
│ ✓ 다양한 관점 확보              │
│ → 프레임워크 다양화 필요        │
│ → 토론 참여 횟수 증가 권장      │
└────────────────────────────────┘
```

#### 구성 요소
- **성장 포인트**: 핵심 피드백 1-2문장
- **강점 2개**: ✓ 녹색 체크마크
- **개선점 2개**: → 파란색 화살표

## 💻 기술 구현

### Chart.js 라이브러리
```html
<script src="https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js"></script>
```

### 데이터 구조
```javascript
// 샘플 데이터에서 지표 가져오기
const metrics = currentSample?.insights?.growthMetrics || {
    depth: 7.5,        // 질문 깊이
    perspective: 8.0,  // 관점 다양성
    structure: 7.0     // 구조화 능력
};
```

### 초기화 함수
```javascript
function initializeCharts() {
    // 1. 점수 텍스트 업데이트
    document.getElementById('depth-score').textContent = `${metrics.depth}/10`;
    
    // 2. 프로그레스 바 애니메이션 (200ms 후)
    setTimeout(() => {
        document.getElementById('depth-bar').style.width = `${metrics.depth * 10}%`;
    }, 200);
    
    // 3. 레이더 차트 생성
    new Chart(radarCtx, { ... });
    
    // 4. 성장 추이 차트 생성
    new Chart(trendCtx, { ... });
}
```

### 호출 시점
```javascript
// Discussion → Write 전환 시
function completeDiscussion() {
    // 화면 전환
    document.getElementById('write-screen').classList.remove('hidden');
    
    // 100ms 후 차트 초기화 (DOM 렌더링 대기)
    setTimeout(() => {
        initializeCharts();
    }, 100);
}
```

## 📊 측정 항목 상세

### 질문 깊이 (Question Depth)
- **측정**: Why 질문, 가정 질문 활용도
- **점수**: 표면 질문(1-3) → 본질 질문(8-10)
- **예시**: "어떻게?" → "왜?"

### 관점 다양성 (Perspective Diversity)
- **측정**: 여러 시각에서 질문
- **점수**: 단일 관점(1-3) → 다면 관점(8-10)
- **예시**: 자신 → 고객 → 시장

### 구조화 능력 (Structuring Ability)
- **측정**: 프레임워크 활용도
- **점수**: 비구조(1-3) → 체계화(8-10)
- **예시**: 무작위 → Head·Heart·Body

### 비판적 사고 (Critical Thinking)
- **측정**: 전제 의심, 검증
- **계산**: 질문 깊이 × 0.9
- **예시**: 수용 → 의심 → 검증

### 창의적 연결 (Creative Connection)
- **측정**: 서로 다른 개념 연결
- **계산**: 관점 다양성 × 0.95
- **예시**: 단절 → 유사 → 창의적 융합

### 실행력 (Execution)
- **측정**: 통찰 → 액션 전환
- **계산**: 구조화 능력 × 1.1
- **예시**: 이론 → 구체적 계획

## 🎨 UI/UX 디테일

### 레이더 차트
```javascript
{
    backgroundColor: 'rgba(59, 130, 246, 0.2)',  // 20% 투명
    borderColor: 'rgb(59, 130, 246)',            // 파란색
    borderWidth: 2,                              // 2px 선
    pointRadius: 4,                              // 기본 점 크기
    pointHoverRadius: 6                          // 호버 시 크기
}
```

### 성장 추이 차트
```javascript
{
    tension: 0.3,        // 곡선 정도
    fill: true,          // 영역 채우기
    pointRadius: 4,      // 점 크기
    pointHoverRadius: 6  // 호버 크기
}
```

### 프로그레스 바
```css
.transition-all {
    transition-property: all;
    transition-duration: 1000ms;
    transition-timing-function: ease-in-out;
}
```

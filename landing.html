RQTDW 학습 프로세스

5단계 체계적 학습으로 효과적인 성장을 경험하세요

📖

1. Read (읽기)

주제를 읽고 이해하며 핵심 내용을 파악합니다. 문제 상황을 명확히 인식하는 것이 시작입니다.

  • • 문제 상황 파악 및 맥락 이해
  • • 핵심 개념과 용어 정리
  • • 배경 지식 활성화
❓

2. Question (질문)

좋은 질문을 만들어 보세요. 15가지 질문 유형을 활용하여 다양한 관점에서 탐구합니다.

  • • 15가지 질문 유형 (기억, 확인, 개념, 관찰 등)
  • • 본질을 파고드는 깊은 질문
  • • 창의적이고 비판적인 사고 촉진
💡

3. Think (사고)

체계적인 사고 프레임워크를 활용하여 깊이 있게 생각하고 분석합니다.

  • • 15가지 사고 프레임워크
  • • 논리적 분석과 비판적 사고
  • • 창의적 문제 해결
💬

4. Discuss (토론)

다양한 관점을 고려하고 토론 가이드를 통해 생각을 확장합니다.

  • • 다양한 관점과 시각 탐색
  • • 토론 가이드와 질문 제시
  • • 협력적 사고와 의견 교환
✍️

5. Write (기록)

배운 내용을 정리하고 기록하여 성장을 추적합니다. 대시보드로 진행 상황을 확인하세요.

  • • 학습 내용 정리 및 기록
  • • 성장 추적 및 대시보드
  • • PDF 보고서 생성

15가지 질문 유형

다양한 질문 유형으로 사고의 폭을 넓히세요

🧠

기억 질문

과거 경험을 회상하여 현재와 미래를 탐구

✅

확인 질문

현재 감정과 생각을 명확히 파악

📚

개념 질문

복잡한 개념을 명확하게 정의하고 구조화

👀

관찰 질문

세심한 관찰로 패턴과 차이점 발견

❤️

감정 질문

감정 반응을 깊이 이해하고 공감 향상

🔍

논리 질문

근거, 증거, 논리적 타당성 탐구

🌊

심화 질문

연쇄 질문으로 사고의 깊이 확장

📖

학습 질문

경험에서 교훈과 통찰 정리

🤔

가정 질문

가상 상황을 설정하여 다양한 가능성 탐색

+ 행동, 의사결정, 공감, 상상, 관계, 관점 질문 등 총 15가지
